# yeehawhub

Python service hub for managing games I host for friends

## General Goal

Log in, create account, add information that relates to you, add yourself to games and servers you like, all while things are secure and locally stored and managed

## Tasks to complete

- General website built
- user accounts created and managable
- accounts used to provide necessary information to enable semi-automatic whitelisting
- heartbeat monitoring for hosted servers
- forums for discussion

## Tech stack

- python
- flask
